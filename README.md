# LoadStock - Sistema PDV Moderno

Sistema de Ponto de Venda (PDV) moderno e completo desenvolvido em Flutter.

## 🚀 Características

- Interface moderna e intuitiva
- Gestão completa de produtos e estoque
- Sistema de vendas eficiente
- Relatórios e análises
- Integração com API REST
- Suporte multiplataforma (Android, iOS, Web, Desktop)

## 📋 Pré-requisitos

- Flutter SDK 3.41.7 ou superior
- Dart 3.11.5 ou superior

## 🔧 Instalação

1. Clone o repositório:
```bash
git clone https://github.com/wegopay/loadstock-app.git
cd loadstock-app
```

2. Instale as dependências:
```bash
flutter pub get
```

3. Execute o aplicativo:
```bash
flutter run
```

## 📁 Estrutura do Projeto

```
lib/
├── core/
│   ├── constants/      # Constantes da aplicação
│   ├── theme/          # Temas e estilos
│   └── utils/          # Utilitários
├── data/
│   ├── models/         # Modelos de dados
│   ├── repositories/   # Repositórios
│   └── services/       # Serviços (API, Storage)
└── presentation/
    ├── screens/        # Telas da aplicação
    ├── widgets/        # Widgets reutilizáveis
    └── providers/      # Gerenciamento de estado
```
